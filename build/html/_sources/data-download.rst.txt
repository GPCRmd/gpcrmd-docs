Simulation report
===================
All files related to the simulations stored in GPCRmd are available for download at the *Simulation report* page. This includes simulation outputs (trajectories, topologies, and coordinates) as well as simulation protocol and starting files. 

The *Simulation report* can be accessed from the *List of simulated systems* tab of our `Search page <https://gpcrmd-docs.readthedocs.io/en/latest/simulation-browser.html>`_ or directly from the *GPCRmd Workbench* page of the simulation, through the *Simulation report and files* link. 

GPCRmd API
===================
In the case that you need to download more than one dynamic files, GPCRmd users have the posibility to use GPCRmd API downloader tool (`GPCRmd API <https://gpcrmd-docs.readthedocs.io/en/latest/api.html>`_) to get a zip file with all the information requested.
